# miniBot QQ频道机器人

## 功能概述
- 实时翻译
- 设置待办事项，到指定时间时群内艾特或发邮箱提醒
- 文心一言 AI 对话
- 随机铯图
- 待补充


## 启动方法
- 进入到 `main.go` 文件所在目录
- 修改 `config.yaml.demo` 为 `config.yaml`，并在其中补充相应配置，注意要符合 yaml 格式要求
- 执行 `go build` 后运行生成的可执行文件或者直接运行 `go run .`
